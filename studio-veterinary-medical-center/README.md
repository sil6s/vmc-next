# Veterinary Medical Center Sanity Studio

Local Sanity Studio for blog posts.

```bash
npm install
npm run dev
```

Open `http://localhost:3333` and log in with the same Sanity account used for project `zk507aly`.

The front-end in `../react-site` reads published `post` documents from the `production` dataset and falls back to static posts if Sanity has no readable posts.
